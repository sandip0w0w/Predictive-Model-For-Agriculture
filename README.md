**About the Project**
* Architected an end-to-end Scikit-Learn Pipeline to streamline data ingestion, feature scaling, and multi-class classification, reducing data leakage and ensuring reproducible workflows.

* Optimized Model Performance by leveraging RandomizedSearchCV for automated hyperparameter tuning, significantly improving F1-scores across diverse soil and climate profiles.

* Validated Statistical Significance using Tukey’s Honest Significant Difference (HSD) test to analyze nutrient overlaps (N, P, K), identifying specific crop clusters with statistically similar profiles to improve model transparency.

* Engineered an Interactive Streamlit Dashboard to provide real-time crop recommendations, featuring dynamic data visualizations and model interpretability tools for non-technical stakeholders.

* Ensured Model Robustness through rigorous evaluation using Confusion Matrices and Classification Reports, specifically targeting the reduction of false positives in high-stakes agricultural recommendations.
